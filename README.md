# Netflix-Data-Analysis
>>Project Overview
This project analyzes the Netflix Movies and TV Shows dataset to understand trends in content type, release years, genres, and countries producing Netflix content. The analysis was performed using Python and data visualization techniques.
>>Objectives
Explore Netflix dataset containing movies and TV shows.
Perform data cleaning and preprocessing.
Analyze trends in Netflix content.
Visualize data using charts and graphs.
>>Tools and Technologies
Python
Jupyter Notebook
Pandas
Matplotlib
Seaborn
Dataset
The dataset used in this project contains information about Netflix titles including:
Title
Type (Movie or TV Show)
Director
Cast
Country
Release Year
Rating
Genre
>>Analysis Performed
Data cleaning and handling missing values
Exploring dataset structure
Analyzing number of Movies vs TV Shows
Identifying top countries producing Netflix content
Studying content release trends over the years
Exploring popular genres on Netflix
>>Key Insights
Movies are more numerous than TV shows on Netflix.
The United States produces the highest amount of Netflix content.
Netflix content increased significantly after 2015.
Drama and International genres are among the most common.
>>Project Files
Netflix_Data_Analysis.ipynb – Jupyter Notebook containing analysis and visualizations
netflix_titles.csv – Dataset used for analysis
>>Conclusion
This project demonstrates basic Exploratory Data Analysis (EDA) skills using Python. It helps understand trends in Netflix content and provides insights through data visualization.## Visualizations
